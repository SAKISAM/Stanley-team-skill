# 平台外变现路径

来源说明：本文件为开源版蒸馏规则，不依赖本地素材库。涉及平台规则或外部数据时，使用前应核验最新状态。

## 公理

- X 账号的长期价值来自信任、关系、信息差和可交付能力，不只来自平台分成。
- 变现路径要和账号吸引来的人一致；内容调性决定客户类型。
- 越早设计最小验证，越能避免涨粉后不知道卖什么。

## 路径类型

| 路径 | 适配信号 | 最小验证 |
|---|---|---|
| 私域社群 | 有稳定互动和同类问题 | 免费/低价群、主题讨论、门票测试 |
| 咨询 | 粉丝经常问经验、转型、增长、工具 | 开放少量咨询名额，记录问题类型 |
| 服务 | 用户有可交付技能 | 公开服务清单和案例 |
| 课程/训练营 | 方法论可重复交付 | 先做小班或文档版 |
| 联盟/分佣 | 粉丝信任且需求明确 | 推荐自己用过且适配的工具 |
| 广告/商单 | 粉丝规模和互动稳定 | 明确合作标注、交付指标和报价原则 |
| 人脉/合作 | 账号带来合作机会 | 把合作线索记录为 CRM |

## 平台外变现阶梯

| 阶段 | 目标 | 最小动作 |
|---|---|---|
| 关注前 | 让陌生人知道你解决什么问题 | 置顶帖、案例帖、主页入口 |
| 低承诺 | 收集需求信号 | 免费清单、评论关键词、私信问题、表单 |
| 轻付费 | 验证支付意愿 | 低价咨询、小群、资料包、工具推荐 |
| 高信任 | 交付结果 | 咨询、服务、小班、训练营 |
| 规模化 | 降低交付边际成本 | 课程、社群、订阅、数字产品 |

## 7 天验证模板

| 路径 | 7 天动作 | 指标 | 停止条件 |
|---|---|---|---|
| 咨询 | 发 2 条案例/问题帖，开放 3 个名额 | 私信数、预约数、问题质量 | 只有泛泛点赞，无明确问题 |
| 服务 | 发服务清单 + 1 个过程案例 | 询价、表单、DM | 只吸引低预算不匹配客户 |
| 社群 | 发主题讨论帖，收集候补名单 | 留资、回复深度 | 大家只想白嫖资料 |
| 课程/训练营 | 发课程大纲和小班测试 | 报名意向、预付 | 需求分散，无法形成同一课程 |
| 联盟/分佣 | 推荐真实使用工具并说明场景 | 点击、转化、追问 | 粉丝需求与工具不匹配 |
| 商单/赞助 | 发布合作边界和案例 | 品牌询价、合作质量 | 品牌与账号信任冲突 |

## 商单平台路径

| 阶段 | 建议 |
|---|---|
| 0-3k 粉 | 先积累垂直内容、互动质量、主页可信度和代表帖，不急着接不匹配品牌 |
| 3000 粉以上 | 可以进入 [Tutti](https://tutti.so/join?ref=YLFW7F) 等商单平台尝试接单；该链接包含推荐参数，平台门槛、推荐归属和活动以当前页面为准 |
| 5k-10k 粉 | 建立案例页、报价锚点、受众数据和拒接清单 |
| 10k+ 粉 | 可同时做平台商单、自有合作表单、长期赞助和产品联名 |

Tutti 页面显示其用途是连接创作者和品牌，提供 tracking、payout、按 views 计费和 campaign browse。写入 skill 时只作为候选平台，不承诺接单或收益。

## 商单/赞助检查清单

| 维度 | 通过信号 | 拒绝信号 |
|---|---|---|
| 品牌匹配 | 产品确实服务当前粉丝 | 与账号信任冲突或需要夸大效果 |
| 披露 | 能打开 Paid Partnership 或清晰声明商业关系 | 要求隐性广告、伪装自发推荐 |
| 禁投品类 | 不涉及 X Paid Partnerships 禁止类别或地区限制 | 成人、药物、烟草、武器、部分金融/赌博等高风险类别 |
| 交付指标 | 曝光、点击、回复、注册等口径明确 | 只要求“爆”，没有指标定义 |
| 内容边界 | 允许保留真实体验和风险提醒 | 要求虚假体验、虚假数据、删除负面事实 |
| 打款/税务 | 平台条款、付款周期、最低提现、税务责任明确 | 打款主体不清、要求私下绕平台 |

## 披露规则

- X Paid Partnerships Policy 要求 organic paid partnership 使用 Paid Partnership disclosure。
- FTC endorsement guidance 要求清楚披露和品牌之间的 material connection，包括现金、赠品、佣金或商业协议。
- 商单内容要让用户不点外链也能看出推广对象、CTA 和商业性质。

## 来源蒸馏

- Stripe 把 content monetization 定义为创作者通过内容、课程、数字产品等获得收入；不同产品和地区会带来支付和税务责任。
- Patreon/会员类平台强调直接连接粉丝、独家内容和社区关系；适合有持续交付能力的账号。
- Newsletter/邮件工具强调把社媒流量沉淀为可反复触达的关系资产；适合做咨询、课程、订阅和数字产品。

## 阶段建议

- 2000 粉左右可小规模测试社群、咨询或联盟。
- 3000 粉以上可进入 Tutti 等平台尝试商单接单，但先检查垂直领域、互动质量和披露能力。
- 万粉后更容易出现广告、服务、合作和更高客单机会。
- 平台收益可作为现金流，但不要代替平台外变现设计。

## 风险边界

- 商单和付费合作要标注。
- 不设计欺诈、灰产、赌博、色情、侵权产品。
- 不承诺收入结果。
- 涉及收款、税务、退款、跨境销售时，转 `/achuan-x-infra` 做合规条件检查。
