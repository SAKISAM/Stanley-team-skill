# 账号方向公理与赛道判断

来源说明：本文件为开源版蒸馏规则，不依赖本地素材库。涉及平台规则或外部数据时，使用前应核验最新状态。

## 公理

- 起号先解决“谁会先关注我”，再解决“我最终要变成什么账号”。
- 账号方向不是一句人设标签，而是身份、受众、内容资产、变现后路四者的可持续组合。
- 新号如果没有强 IP 背书，泛流量更容易完成冷启动；垂直信任可以在初始流量和反馈出现后逐步收束。
- 做号前不要只问“什么内容会火”，要先问“这个方向未来靠什么赚钱”。

## 方向判断

| 判断项 | 好信号 | 坏信号 |
|---|---|---|
| 身份 | 有真实经历、正在经历的变化、可被持续观察的过程 | 只有空泛头衔或复制别人标签 |
| 受众 | 能说清楚具体谁会因为你少走弯路 | 面向“所有人” |
| 内容资产 | 有可持续素材、案例、观点或实验过程 | 只能靠当天热点硬凑 |
| 变现后路 | 咨询、社群、服务、课程、工具、联盟或广告至少有一条可能路径 | 完全不知道未来承接什么需求 |

## 操作规则

- 让用户先列背景、经历、技能、正在转型的问题和可帮别人节省的时间。
- 用对标账号拆三件事：靠什么内容吸粉、靠什么产品或服务变现、这条路径用户能不能长期做。
- 生成方向候选时，每个候选必须同时给出“涨粉入口”和“信任承接”。

## 路由

- 方向确认后进入 `/achuan-x-profile`。
- 用户想先测内容题材时进入 `/achuan-x-content-system`。
- 用户已经有产品或服务时进入 `/achuan-x-monetization-path`。
